# Tippo - The Language of Gods

A language that is made just for fun. It has riveting features such as:
- [x] Integers
- [x] Basic Arithmetic
- [x] Const and Mutable variables
- [ ] Scopes
- [ ] Booleans & chars
- [ ] Array
- [ ] Function
- [ ] More :O
